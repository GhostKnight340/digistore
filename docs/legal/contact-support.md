# Contact & Support

_Dernière mise à jour : 30 juin 2026_

Notre équipe est là pour vous accompagner avant, pendant et après votre achat. Que vous ayez une question sur un produit, un paiement ou une commande, vous obtenez une réponse claire et rapide.

## Nous contacter

Vous pouvez joindre notre équipe support via les canaux mis à disposition sur la plateforme. Pour un traitement plus rapide, indiquez systématiquement votre **numéro de commande** ainsi qu'une description précise de votre demande.

Si vous disposez d'un compte client, vous retrouvez l'ensemble de vos commandes et de leur statut directement depuis votre tableau de bord sécurisé — c'est souvent le moyen le plus rapide d'obtenir l'information recherchée.

## Suivre votre commande

Vous gardez le contrôle sur vos achats à tout moment :

- **Par email** : vous recevez la confirmation, le statut et la livraison de vos codes directement dans votre boîte de réception.
- **Sur le site** : connectez-vous à votre compte pour consulter votre historique de commandes, ou utilisez le suivi de commande pour vérifier l'état d'un achat en tant qu'invité.

## Comment pouvons-nous vous aider

Notre équipe support intervient notamment pour :

- répondre à vos questions sur un produit, une région ou une compatibilité avant l'achat ;
- vous renseigner sur le statut d'une commande en cours de traitement ;
- vous accompagner en cas de code invalide ou inutilisable ;
- traiter les demandes de remboursement éligibles ;
- vous aider à gérer votre compte et votre historique de commandes.

## Questions fréquentes

### La livraison est-elle rapide ?

Les produits à traitement automatique sont livrés rapidement après la confirmation de votre paiement. Les produits nécessitant une vérification manuelle sont livrés dans les meilleurs délais une fois cette vérification effectuée. Dans tous les cas, vous êtes informé par email et sur le site.

### Pourquoi un justificatif de paiement peut-il m'être demandé ?

Selon le moyen de paiement choisi, un justificatif nous permet de confirmer votre transaction et de sécuriser votre commande. Cette étape protège à la fois nos clients et la plateforme.

### Pourquoi ma commande est-elle en cours de vérification ?

Pour prévenir la fraude, certaines commandes font l'objet d'une vérification complémentaire. Cette étape peut allonger légèrement le délai de traitement. Si aucun problème n'est détecté, votre commande est livrée normalement.

### Mon code ne fonctionne pas, que faire ?

Contactez notre support en précisant votre numéro de commande et le code concerné. Nous vérifions la situation et, si le code est invalide de notre fait, nous le remplaçons ou vous remboursons, conformément à notre [Politique de Remboursement](./remboursement.md).

### Puis-je acheter sans créer de compte ?

Oui. Vous pouvez commander en tant qu'invité. Créer un compte vous donne toutefois accès à votre historique de commandes, au suivi de vos achats et à un tableau de bord sécurisé.
